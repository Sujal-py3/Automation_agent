# Automation Agent 

Automation Agent is an AI-powered assistant that works entirely through WhatsApp, allowing users to manage emails and automate basic tasks using natural language.

## Features

- Send emails via WhatsApp  
- Set reminders (coming soon)  
- Reply to emails (coming soon)  
- Custom automation tasks (coming soon)  

## How It Works

### Initial Setup
- Send "Hi", "Hello", or "Hey" on WhatsApp
- Receive a menu with available actions

### Email Composition Flow
1. Greet to receive menu
2. Select "Send Email"
3. Provide recipient email address
4. Describe intent in natural language
5. Review generated draft
6. Confirm with "Send", "Cancel", or "Edit"
